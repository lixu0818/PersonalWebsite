Project I plan to do:

“TB or not TB: detect tuberculosis before symptoms hit”

Solution:

1. Data collection and understanding: query and combine the following data from TB patient or healthy individuals: a, electronic health record b, daily activity c, personal fitness data from wearables. Plot collected data to develop initial insights and identify data quality issues.

2. Data preparation: Use pandas to frame data into clean format that can be fed into modeling tools.

3. Modeling: Split data into train and test sets. Train and test different ML models such as random forest, SVM, or Neural net. Pick the model gives best prediction accuracy, or combine different prediction models to get better performance.

4. Dimension reduction: algorithms such as PCA, LDA, random projections may be used to reduce the dimension of data in order to improve time performance.

5. Model evaluation: thoroughly test the final model; iterate the learning process to improve prediction accuracy.

6. Presenting the results as a summary report. Implement a prediction software that can repeat the whole data mining and disease prediction process.

Objective: I hope to leverage the power of data analysis to establish an early detection model for tuberculosis. Once established, similar workflow may be used for finding early detection model of other latent diseases such as hepatitis B, heart diseases
